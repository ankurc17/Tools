# VarFreqAnalyzer
A comprehensive Python GUI tool for population-scale variant frequency analysis.

This repository includes core Python script and Google Drive link having a Windows .exe file, and a Linux executable will be provided in the future for improved performance.


## Installation

### Windows

1. Download the exe file.
2. Run the exe directly â€” no Python installation needed.
3. Google drive Link - https://drive.google.com/file/d/1MXh9z0Oznn9mmzT8mMzsg5ldAkyWuCPC/view?usp=sharing
   ### Usage
   Launch the app.
   
   Select the VCF repository directory.
   
   Enter genomic coordinates in the format: chr1:10000-20000.
   
   Optionally enter REF and ALT alleles to calculate internal variant frequency.
   
   Click Run Analysis.
